# worklogs

Archived research memory.

- `attempts/` contains detailed records for historical failed or weak research
  directions. Do not delete or rewrite these files during harness work.
- `../prior_attempts.md` is the compact index agents should read by default.
- New detailed attempt files are only added after code has run and produced
  benchmark evidence.

The active harness does not read this directory. It exists to prevent the agent
from rediscovering old dead ends.
