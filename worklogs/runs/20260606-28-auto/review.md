## Reasoning

PRAR's structural identity is a two-part composition: (1) a **regime index computed from the policy's own action-distribution shape** (entropy-bin × top-prob-bin × top-2-gap-bin, fully internal to the agent), and (2) an **antisymmetric within-episode-position residual** `w(t/T) = 2t/T − 1` applied to per-channel deviations from each episode's mean firing rate. Neither element appears in any prior attempt #1–35 or in any named disqualifier family. The regime axis is populated on every decision step regardless of what the environment returns — it does not require observation-hash collisions (ruling out the FED/CEC/TPP/TRAC/CSD/ARP/PCR bootstrap-wall family), it does not require reward correlation before the first rewarded trajectory (ruling out KERNEL/OPP/ACFD/ACCD/PICAV/CHX), and it does not rely on any learned projection against terminal cumulants (ruling out PHI/PFA's regressor-bootstrap collapse). The improvement operator is the coordinate-wise Pareto-non-dominance count applied to the k-vector `F[r_t, a, :]`, which is invariant under per-channel monotone rescaling and thus is not equivalent to any `w^T r` scalarization. The nearest alive-weak candidates KTAC and DPC are correctly distinguished: KTAC uses k-means context clusters and Kemeny rank-aggregation; DPC uses trajectory-pair divergence-state hashes and sign-votes — neither uses policy-distribution-shape as the indexing axis or antisymmetric position weighting as the residual structure.

All required-candidate-shape slots are substantively filled: the experience object, primitive definition with update rule, improvement operator with a precise logit-nudge formula, execution rule, per-channel vector feedback rule, zero-counterfactual rollout discipline, nearest-neighbor novelty audit, five concrete predicted failure modes with diagnostics, named side-information channel, and a precise monotonic improvement claim (fixed-F Pareto-dominance count monotonicity by construction, with the regime-stationarity caveat stated). The hypothesis's own falsifier is specific and testable: if regime-occupancy collapses to a single regime on both vector envs or cell populations stay below 5 episodes per cell, the primitive is predicted to be silent.

## Risks the Engineer should be aware of

- The hypothesis acknowledges failure mode (c): the universal step-penalty channel cancels when `v_t[m] = const` within an episode, leaving `k_eff = 1` (only the terminal-reward channel). On DST/RG this means the Pareto vote collapses to a scalar comparison on the single surviving channel — technically not a disqualifier (it is still the antisymmetric-residual primitive on that channel), but the practical signal is thin until enough rewarded episodes accumulate to distinguish action-regime cells. The hypothesis should diagnose this with the per-channel non-zero-fraction metric and set `α = 0` below the N[r,a] < 5 gate.
- Regime feedback loop (failure mode (e)): nudging logits changes the policy distribution, which changes the regime, which changes F readouts. At large `α` this can cause oscillation. The stated bound `α ≤ 0.05 / k` should be enforced in the implementation; the Engineer should verify the regime-occupancy histogram is not drifting monotonically toward a single bin during training.
- On envs with a tiny discrete action space (DoorKey: 7 actions), the Pareto-non-dominance count `D_+(a) − D_-(a)` has low dynamic range (max value ≈ 6). The logit nudge `α · D` may be negligible relative to the base policy's logits if `α` is not tuned upward; the Engineer should confirm the nudge magnitude is at least comparable to one logit standard deviation.
