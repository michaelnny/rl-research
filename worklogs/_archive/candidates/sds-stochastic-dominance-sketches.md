SDS (Stochastic-Dominance Sketches) maintains per-(state-cluster, action, channel) compressed quantile sketches (T-digest / sorted reservoir, size B=64) of empirical continuation-cumulant distributions `F_{s,a,m}(x) = Pr[c_m(τ_T) − c_m(τ_t) ≤ x | cluster(o_t)=s, a_t=a]`. At decision time, the improvement operator nudges logits by `α·(D+(a) − D-(a))` where D+/D- count actions that strictly first-order stochastically dominate / are dominated by `a` on every channel simultaneously — the SD-conjunction partial order on CDFs, not Pareto-mean comparison. Side information: vector diagnostics (per-channel continuation deltas) + transition geometry (observation embedding for k-means cluster assignment). The mechanism is structurally novel: unlike distributional RL it never bootstraps or takes a Bellman backup of the distribution and never extracts a scalar from it; unlike VCC it uses per-channel CDF shape rather than Pareto-front of mean vectors.

The blocking issue: the "≥ 2 non-degenerate channels" gate (the structural guard that prevents collapsing to scalar step-penalty comparison) is too restrictive for the substrate's terminal-only reward channels. Both DST and RG have reward channels that fire only at episode termination; the gate never allows ≥ 2 simultaneously non-degenerate sketch cells within the 120 s budget, so the operator is permanently silent. Scores: DST 99.0 vs random 194.0; RG 0.011 vs random 1.331 — both below random, matching the silent-operator pattern. The fix is a graduated gate: allow single-channel SD with reduced alpha during the bootstrap window (when only the step-penalty channel has non-degenerate sketches), escalating to full multi-channel SD conjunction once ≥ 2 channels are populated. This preserves the anti-rebadge structural property while allowing early activation. Alternatively, initialize reward-channel sketches with a small prior spread so they are non-degenerate from episode 1 (would require careful calibration to avoid biasing the SD comparison before real samples arrive).
