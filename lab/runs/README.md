# Runs

Each run is a directory `<run_id>/` with the contract layout from
`docs/contract.md`. The Operator and Researcher write here; the Curator reads.

`run_id` format: `NNNN-thread-slug` (zero-padded 4-digit sequence + thread
fragment). Allocated by `rl_research.contract.next_run_id`.

This README intentionally short — see `docs/contract.md` for the full layout.
